전기기초 및 자동제어/시퀀스 핵심 50제

공조냉동기계기능사 필기 단골 문제 정복
⚡ 틈새 학습 팁: 전기/제어 파트는 까다로운 수식보다 과년도 기출문제 문장 그대로 출제되는 경향이 매우 강합니다. 눈으로 질문을 읽고 정답 키워드를 바로바로 꺼내는 훈련을 해보세요!

📌 PART 1. 전기 이론 및 계측 기초 (15문)

  • Q1. 전류의 흐름을 방해하는 저항(R)에 전류(I)가 흐를 때 발생하는 열량을 구하는 법칙은?
    정답 확인하기
    👉 정답: 줄의 법칙 (Joule's law, H = 0.24 * I² * R * t)
  • Q2. 회로에 흐르는 전류의 크기는 전압에 비례하고 저항에 반비례한다는 기본 법칙은?
    정답 확인하기
    👉 정답: 옴의 법칙 (Ohm's law, I = V / R)
  • Q3. "회로망의 임의의 한 점에 흘러들어오는 전류의 총합과 흘러나가는 전류의 총합은 같다"는 법칙은?
    정답 확인하기
    👉 정답: 키르히호프의 제1법칙 (전류 법칙)
  • Q4. 회로에서 전압을 측정할 때 전압계는 부하와 어떤 방식으로 연결해야 하는가?
    정답 확인하기
    👉 정답: 병렬 연결
  • Q5. 회로에서 전류를 측정할 때 전류계는 부하와 어떤 방식으로 연결해야 하는가?
    정답 확인하기
    👉 정답: 직렬 연결
  • Q6. 교류 회로에서 실제로 유효하게 일을 하는 전력의 단위는?
    정답 확인하기
    👉 정답: W (와트) 또는 kW (유효전력)
  • Q7. 피상전력에 대한 유효전력의 비율을 뜻하며, 전기를 얼마나 효율적으로 썼는지 나타내는 지표는?
    정답 확인하기
    👉 정답: 역률 (Power factor)
  • Q8. 모터 등 코일 성분에 의해 전압보다 전류의 위상이 뒤지는(늦어지는) 역률을 무엇이라 하는가?
    정답 확인하기
    👉 정답: 지상 역률 (부하가 유도성일 때 발생)
  • Q9. 냉동기 모터 절연 저항을 측정할 때 주로 사용하는 계측기의 명칭은?
    정답 확인하기
    👉 정답: 메거 (Megger / 절연저항계)
  • Q10. 전선의 피복을 벗기지 않고 전선에 집게처럼 물려서 전류를 안전하게 측정할 수 있는 계측기는?
    정답 확인하기
    👉 정답: 훅온 메터 (Clamp Meter / 클램프 미터)
  • Q11. 콘덴서(커패시터)가 가진 특성으로, 전기를 축적할 수 있는 능력을 뜻하는 용어와 단위는?
    정답 확인하기
    👉 정답: 정전용량, 단위는 F (패럿)
  • Q12. 유도성 부하(모터) 때문에 낮아진 역률을 개선하기 위해 부하와 병렬로 연결하는 부품은?
    정답 확인하기
    👉 정답: 전력용 콘덴서 (진상 콘덴서)
  • Q13. 2개의 서로 다른 금속선 양단을 접합하고 양쪽에 온도차를 주면 열기전력이 발생하는 현상은?
    정답 확인하기
    👉 정답: 제베크 효과 (Seebeck effect / 열전대 온도계의 원리)
  • Q14. 반대로 두 금속의 접합점에 전류를 흘리면 한쪽은 열을 흡수하고 다른 쪽은 열을 방출하는 현상은?
    정답 확인하기
    👉 정답: 펠티에 효과 (Peltier effect / 전자냉동의 원리)
  • Q15. 전류가 흐르는 도체를 자기장 속에 놓을 때 도체가 받는 힘의 방향을 결정하는 법칙은?
    정답 확인하기
    👉 정답: 플레밍의 왼손 법칙 (모터/전동기의 원리)

📌 PART 2. 자동제어 계통 및 블록선도 (10문)

  • Q16. 제어 결과를 입력 측으로 다시 보내어 목표값과 비교하여 오차를 수정하도록 하는 제어 방식은?
    정답 확인하기
    👉 정답: 피드백 제어 (Feedback Control / 폐루프 제어)
  • Q17. 외부 요인에 의해 제어량을 변화시켜 피드백 제어 시스템의 상태를 교란하는 원치 않는 입력 신호는?
    정답 확인하기
    👉 정답: 외란 (Disturbance)
  • Q18. 피드백 제어계의 기본 블록선도에서 '제어 요소'는 무엇과 무엇으로 구성되는가?
    정답 확인하기
    👉 정답: 조절부 + 조작부
  • Q19. 제어 요소가 제어 대상(플랜트)에 주는 입력 신호로, 상태를 바꾸기 위해 물리량을 가하는 신호는?
    정답 확인하기
    👉 정답: 조작량
  • Q20. 피드백 제어계에서 출력 신호를 감지하여 입력 측의 비교부로 보내주는 역할을 하는 전송 장치는?
    정답 확인하기
    👉 정답: 검출부 (피드백 요소)
  • Q21. 선박의 방향 제어, 항공기의 유도 제어 등 '위치, 방위, 자세'를 제어량으로 하는 자동제어 분류는?
    정답 확인하기
    👉 정답: 서보 기구 (Servo Mechanism)
  • Q22. 공조냉동 설비에서 '온도, 습도, 압력, 유량' 등 공업 프로세스 상태량을 제어량으로 하는 제어는?
    정답 확인하기
    👉 정답: 프로세스 제어 (Process Control)
  • Q23. 회전속도, 전압, 전류, 주파수 등을 제어량으로 하며 응답 속도가 매우 빠른 자동제어 분류는?
    정답 확인하기
    👉 정답: 자동 조정 (Automatic Regulation)
  • Q24. 무인 열차나 엘리베이터 동작처럼 목표값이 미리 정해진 시간 스케줄에 따라 변하는 제어 방식은?
    정답 확인하기
    👉 정답: 프로그램 제어
  • Q25. 유도미사일 추적이나 레이더 제어처럼 목표값이 시간에 따라 예측할 수 없이 임의로 변하는 제어 방식은?
    정답 확인하기
    👉 정답: 추치 제어 (Follow-up Control)

📌 PART 3. 제어 동작 특성 (P, I, D 동작) (10문)

  • Q26. 오차의 크기에 비례하여 조작량을 변화시키는 가장 단순한 제어로, 구조는 간단하나 오차가 완전히 없어지지 않고 남는 제어 동작은?
    정답 확인하기
    👉 정답: 비례 동작 (P 동작)
  • Q27. 비례 동작(P 동작)에서 제어 결과가 목표값에 완벽히 도달하지 못하고 약간의 차이를 두고 평형을 이루는 '잔류 오차'를 뜻하는 용어는?
    정답 확인하기
    👉 정답: 오프셋 (Offset)
  • Q28. 오차를 누적 적분하여 조작량을 만듦으로써 비례 동작의 단점인 '잔류 오차(오프셋)를 완전히 제거'할 수 있는 제어 동작은?
    정답 확인하기
    👉 정답: 적분 동작 (I 동작 / PI 제어로 결합하여 많이 사용)
  • Q29. 오차가 커지는 속도(변화율)를 감지하여 미리 조작량을 조절하는 제어로, 제어계의 '속응성(빠른 반응)'을 개선하는 동작은?
    정답 확인하기
    👉 정답: 미분 동작 (D 동작)
  • Q30. 미분 동작(D 동작)은 단독으로 제어계를 구성할 수 있는가?
    정답 확인하기
    👉 정답: 불가능하다 (반드시 P 또는 PI 동작과 결합하여 PD, PID 형태로만 사용 가능)
  • Q31. 잔류 오차가 없고 간헐적인 외란에 대한 감쇄 특성이 우수하며, 속응성까지 확보한 가장 이상적인 제어 동작 조합은?
    정답 확인하기
    👉 정답: 비례 적분 미분 동작 (PID 동작)
  • Q32. 조작량이 불연속적으로 최댓값(ON)과 최솟값(OFF)의 두 단계 값만 가지는 가정용 냉장고나 에어컨에 자주 쓰이는 제어 동작은?
    정답 확인하기
    👉 정답: 2위치 동작 (ON-OFF 동작)
  • Q33. 불연속 제어 동작 중 ON-OFF 제어의 빈번한 스위칭으로 인한 기기 마모를 줄이기 위해 설정하는 동작의 시간적/물리적 간격은?
    정답 확인하기
    👉 정답: 동작 틈새 (차동 / 디퍼렌셜 - Differential)
  • Q34. 입력 신호가 조작 장치에 전달될 때까지 걸리는 시간적 지연이나 물리적 거리에 의한 시간 지연을 뜻하는 제어 용어는?
    정답 확인하기
    👉 정답: 낭비 시간 (부동 시간 / Dead Time)
  • Q35. 자동제어 밸브 중 전기 신호를 받아 내부의 모터 구동 축을 직접 회전시켜 유량을 제어하는 대표적인 조작 기기는?
    정답 확인하기
    👉 정답: 전동 밸브 (Motor 가동 밸브)

📌 PART 4. 시퀀스 제어 회로 및 부품 (15문)

  • Q36. 미리 정해진 순서나 조건에 따라 제어의 각 단계를 차례로 진행시켜 나가는 연속적인 순차 제어 방식은?
    정답 확인하기
    👉 정답: 시퀀스 제어 (Sequence Control / 개루프 제어 계통에 속함)
  • Q37. 시퀀스 제어 부품 중 평소에는 열려(Off) 있다가, 외부에서 힘을 가하거나 조작하면 닫히는(On) 접점의 명칭은?
    정답 확인하기
    👉 정답: A 접점 (메이크 접점 / 평상시 열린 접점)
  • Q38. 평소에는 닫혀(On) 있다가 외부에서 조작을 가하면 회로를 차단(Off)하도록 열리는 접점의 명칭은?
    정답 확인하기
    👉 정답: B 접점 (브레이크 접점 / 평상시 닫힌 접점)
  • Q39. 수동 푸시버튼 스위치에서 손을 때더라도, 릴레이(계전기)의 자체 A접점을 이용하여 제어 회로의 통전 상태를 계속 유지시키는 핵심 회로는?
    정답 확인하기
    👉 정답: 자기 유지 회로 (Self Holding Circuit)
  • Q40. 두 개의 회로가 동시에 동작하여 사고가 나는 것을 막기 위해, 상대방이 동작 중일 때는 내 회로가 절대 켜지지 않도록 서로의 B접점으로 묶어두는 안전 회로는?
    정답 확인하기
    👉 정답: 인터록 회로 (Interlock Circuit / 상호 결쇄 회로)
  • Q41. 모터에 과부하가 걸려 과전류가 지속적으로 흐를 때, 발생한 열에 의해 바이메탈이 굽어지면서 회로를 차단해 모터 소손을 방지하는 보호 부품은?
    정답 확인하기
    👉 정답: 열동계전기 (THR / Overload Relay)
  • Q42. 작은 전류 신호로 전자석을 소자/여자시켜 큰 전류가 흐르는 주회로의 접점을 개폐할 수 있도록 만든 대표적인 시퀀스 전환 부품은?
    정답 확인하기
    👉 정답: 전자기계전기 (릴레이 / Relay)
  • Q43. 대용량 압축기 모터의 동력 전원을 직접 켜고 끄기 위해 사용되는, 전자석과 강력한 주접점을 내장한 동력 제어용 기기는?
    정답 확인하기
    👉 정답: 전자접촉기 (MC / Magnetic Contactor)
  • Q44. 전기 입력 신호가 들어온 즉시 동작하지 않고, 설정된 일정 시간이 지난 후에 접점이 닫히거나 열리도록 지연 특성을 가진 계전기는?
    정답 확인하기
    👉 정답: 타이머 (한시 계전기 / Timer)
  • Q45. 타이머 접점 기호 중 한시 동작 순시 복귀(On-Delay) 접점을 뜻하는 기호 형태적 특징은?
    정답 확인하기
    👉 정답: 접점 깃대 끝 모양이 삼각 모양(삼각형)으로 되어 있음
  • Q46. 논리 회로 중 "두 개 이상의 입력 신호가 모두 1(On)일 때만 출력 신호가 1(On)이 되는" 일명 '직렬 회로'에 해당하는 논리 게이트는?
    정답 확인하기
    👉 정답: AND 회로 (논리곱)
  • Q47. "입력 신호 중 어느 하나라도 1(On)이 되면 출력 신호가 1(On)이 되는" 일명 '병렬 회로'에 해당하는 논리 게이트는?
    정답 확인하기
    👉 정답: OR 회로 (논리합)
  • Q48. 입력 신호와 완전히 반대되는 출력 신호를 내보내는 회로로, 일명 인버터 또는 B접점 회로와 같은 역할을 하는 논리 회로는?
    정답 확인하기
    👉 정답: NOT 회로 (논리 부정)
  • Q49. 유접점 시퀀스 제어 회로와 비교했을 때, 반도체 소자를 이용한 '무접점 시퀀스 제어'의 가장 큰 장점은?
    정답 확인하기
    👉 정답: 동작 속도가 빠르고 수명이 반영구적이다 (기계적 마모가 없음)
  • Q50. 공장 자동화나 공조 설비 제어에 널리 사용되며, 컴퓨터 프로그램을 수정하는 방식으로 제어 시퀀스를 자유롭게 바꿀 수 있는 범용 전자 제어 장치는?
    정답 확인하기
    👉 정답: PLC (Programmable Logic Controller)
